# Structural Fatigue and Metastable Closure in Runtime Monitoring

This repository contains the code, mechanisation notes, and reproducibility artefacts for the paper:

**Structural Fatigue and Metastable Closure in Runtime Monitoring**

## Contents

- `bin/` — executable entry points and demonstrations.
- `lib/` — implementation modules.
- `formal/` — formal proof artefacts.
- `test/` — tests.
- `viz/` — visualisation scripts.
- `experiments/` — benchmark outputs, sweep summaries, trajectories, and rupture certificate examples.
- `mechanisation.md` — notes on the formal mechanisation.
